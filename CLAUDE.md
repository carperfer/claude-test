# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Propósito

Proyecto de prueba para explorar las capacidades de Claude Code: velocidad de desarrollo, generación de código, refactoring, testing y cualquier otra funcionalidad del asistente.

## Objetivo

Experimentar con flujos de trabajo asistidos por Claude — desde scaffolding inicial hasta tareas complejas de ingeniería — para evaluar cómo integrar Claude Code en el ciclo de desarrollo habitual.
